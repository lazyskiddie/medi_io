Capabilities

Built for real-world lab reports

Python backend handles all the heavy lifting — OCR, preprocessing, batch jobs, model training.

🐍

Python server-side OCR

Tesseract + Pillow preprocessing runs on the server. Upload 426 images — no browser crash, no memory limit.

⚡

Background batch jobs

Upload hundreds of Kaggle images at once. Server processes them in the background while you watch live progress.

🗄️

SQLite storage

Everything in one persistent file — training data, user uploads, model weights. No external database needed.

✏️

Admin review & edit

After OCR, review each image's extracted values. Edit wrong numbers, delete bad rows, then approve to training.

🧠

TF.js neural network

Autoencoder trained in browser on server data. Deployed weights served to all users — ML scoring on every report.

🔒

Private by design

Images processed server-side and discarded. Only anonymized numeric values stored in SQLite.
